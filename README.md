## Supported Distributions

- Arch Linux / Manjaro
- Debian / Ubuntu
- Kali Linux
- Parrot OS
- Fedora
- OpenSUSE 

## Setup 

1. Clone the repository:
```bash
git clone https://github.com/Not-Tushar/Tornetwork.git
cd Tornetwork
```

2. Make the installation script executable:
```bash
chmod +x setup.sh
```

3. Run the installation script:
```bash
./setup.sh
```
